# Ayman Ibrahim | Data Engineer Portfolio

A professional portfolio website built with React, Tailwind CSS, and Lucide Icons. 
This project is featuring a responsive design, dark mode, and smooth animations.

## 🚀 Features

- **Modern UI/UX:** Clean, professional design with glassmorphism effects.
- **Dark Mode Support:** Automatically detects system preference with a manual toggle.
- **Responsive:** Fully optimized for mobile, tablet, and desktop devices.
- **Smooth Animations:** Intersection Observer-based fade-in effects for sections.
- **Interactive Elements:** Dynamic navigation, hover states, and contact form UI.

## 🛠️ Tech Stack

- **Frontend Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Typography:** Inter & Poppins (via Google Fonts)

## 📂 Project Structure

- `components/`: Reusable React components (Hero, Projects, Experience, etc.).
- `components/ui/`: UI-specific components (FadeIn, SectionHeading).
- `types.ts`: TypeScript interfaces for type safety.
- `App.tsx`: Main application component.

## 🏃‍♂️ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/portfolio.git
   ```

2. **Navigate to project directory**
   ```bash
   cd portfolio
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

## 🌍 Deployment using GitHub Pages

1. Go to your repository **Settings**.
2. Click **Pages** in the left sidebar.
3. Under **Branch**, select `main` and click **Save**.
4. Your site will be live at `https://yourusername.github.io/repo-name`.

## ⚠️ Important Note
